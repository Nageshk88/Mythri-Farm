# Mytri Farm — Official Website

A fully responsive, multi-page website for **Mytri Farm** — a farm destination near Ras Al Khaimah Airport, UAE, offering staycations, outdoor events, and indoor gatherings.

---

## Live Pages

| Page | File | Description |
|------|------|-------------|
| Home | `mythri-farms.html` | Full landing page with all sections |
| Gallery | `gallery.html` | Photo gallery with category filters & lightbox |
| Redirect | `index.html` | Auto-redirects to the main landing page |

---

## About the Farm

Mytri Farm is a 4-acre destination near RAK Airport with three bookable zones:

| Zone | Capacity | Highlights |
|------|----------|------------|
| Staycation Zone | Up to 30 overnight guests | 5 bedrooms, swimming pool, kitchen, BBQ |
| Outdoor Event Ground | Up to 500 guests | Stage space, sound-ready, flexible seating |
| Indoor Gathering Venue | 70–100 guests | Covered, AV-compatible, catering-ready |

## Features

- Fully responsive design (mobile, tablet, desktop)
- Sticky mobile CTA bar (WhatsApp + Call)
- FAQ accordion
- Enquiry form with WhatsApp redirect
- Photo gallery with category filters and lightbox
- Google Fonts (Playfair Display + DM Sans)
